حلقه while در پایتون

پایتون دو دستور حلقه اولیه دارد:

  • حلقه‌های while
  • حلقه‌های for

حلقه while

با استفاده از حلقه while می‌توانیم مجموعه‌ای از دستورات را تا زمانی که یک شرط درست است اجرا کنیم.

مثال

مقدار i را تا زمانی که کمتر از 6 است چاپ کنید:

i = 1
while i < 6:
  print(i)
  i += 1

توجه: فراموش نکنید که i را افزایش دهید، در غیر این صورت حلقه برای همیشه ادامه خواهد یافت.

حلقه while نیاز دارد که متغیرهای مربوطه آماده باشند. در این مثال ما نیاز داریم که یک متغیر اندیس‌گذاری، i، را تعریف کنیم که آن را روی 1 تنظیم می‌کنیم.

آزمون 1: درک حلقه while

خروجی کد زیر چیست؟

i = 1
while i < 4:
  print(i)
  i += 1

دستور break

با استفاده از دستور break می‌توانیم حلقه را حتی اگر شرط while درست باشد متوقف کنیم.

مثال

هنگامی که i برابر با 3 است از حلقه خارج شوید:

i = 1
while i < 6:
  print(i)
  if i == 3:
    break
  i += 1

آزمون 2: درک دستور break

خروجی کد زیر چیست؟

i = 1
while i < 5:
  if i == 3:
    break
  print(i)
  i += 1

دستور continue

با استفاده از دستور continue می‌توانیم تکرار فعلی را متوقف کرده و با تکرار بعدی ادامه دهیم.

مثال

اگر i برابر با 3 باشد به تکرار بعدی بروید:

i = 0
while i < 6:
  i += 1
  if i == 3:
    continue
  print(i)

آزمون 3: درک دستور continue

خروجی کد زیر چیست؟

i = 0
while i < 5:
  i += 1
  if i == 2:
    continue
  print(i)

دستور else

با استفاده از دستور else می‌توانیم یک بلوک کد را یک بار اجرا کنیم زمانی که شرط دیگر درست نیست.

مثال

یک پیام را یک بار چاپ کنید زمانی که شرط نادرست است:

i = 1
while i < 6:
  print(i)
  i += 1
else:
  print("i دیگر کمتر از 6 نیست")

آزمون 4: درک دستور else در حلقه while

خروجی کد زیر چیست؟

i = 1
while i < 3:
  print(i)
  i += 1
else:
  print("حلقه تمام شد")
