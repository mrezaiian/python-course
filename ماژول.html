ماژول‌های پایتون

ماژول چیست؟

یک ماژول را به عنوان یک کتابخانه کد در نظر بگیرید.

یک فایل حاوی مجموعه‌ای از توابع که می‌خواهید در برنامه خود قرار دهید.

ایجاد یک ماژول

برای ایجاد یک ماژول، کافی است کدی را که می‌خواهید در یک فایل با پسوند .py ذخیره کنید:

مثال

این کد را در فایلی به نام mymodule.py ذخیره کنید:

def greeting(name):
  print("Hello, " + name)

استفاده از یک ماژول

اکنون می‌توانیم از ماژولی که ایجاد کردیم با استفاده از دستور import استفاده کنیم:

مثال

ماژول mymodule را import کنید و تابع greeting را فراخوانی کنید:

import mymodule

mymodule.greeting("Jonathan")

توجه: هنگام استفاده از یک تابع از یک ماژول، از این نحو استفاده کنید: module_name.function_name

متغیرها در ماژول

ماژول می‌تواند شامل توابع باشد، همانطور که قبلاً توضیح داده شد، اما همچنین می‌تواند شامل متغیرهایی از انواع مختلف (آرایه‌ها، دیکشنری‌ها، اشیاء و غیره) باشد:

مثال

این کد را در فایل mymodule.py ذخیره کنید:

person1 = {
  "name": "John",
  "age": 36,
  "country": "Norway"
}

مثال

ماژول mymodule را import کنید و به دیکشنری person1 دسترسی پیدا کنید:

import mymodule

a = mymodule.person1["age"]
print(a)

نام‌گذاری یک ماژول

می‌توانید فایل ماژول را به هر نامی که دوست دارید نام‌گذاری کنید، اما باید پسوند فایل .py داشته باشد.

تغییر نام یک ماژول

می‌توانید هنگام import کردن یک ماژول، با استفاده از کلیدواژه as یک نام مستعار ایجاد کنید:

مثال

یک نام مستعار برای mymodule به نام mx ایجاد کنید:

import mymodule as mx

a = mx.person1["age"]
print(a)

ماژول‌های داخلی

چندین ماژول داخلی در پایتون وجود دارد که می‌توانید هر زمان که خواستید آن‌ها را import کنید.

مثال

ماژول platform را import کرده و از آن استفاده کنید:

import platform

x = platform.system()
print(x)

استفاده از تابع dir()

یک تابع داخلی برای فهرست کردن تمام نام‌های تابع (یا نام‌های متغیر) در یک ماژول وجود دارد. تابع dir():

مثال

تمام نام‌های تعریف شده متعلق به ماژول platform را فهرست کنید:

import platform

x = dir(platform)
print(x)

توجه: تابع dir() را می‌توان در تمام ماژول‌ها، از جمله ماژول‌هایی که خودتان ایجاد می‌کنید، استفاده کرد.

Import از ماژول

می‌توانید با استفاده از کلیدواژه from فقط بخش‌هایی از یک ماژول را import کنید.

مثال

ماژول mymodule دارای یک تابع و یک دیکشنری است:

def greeting(name):
  print("Hello, " + name)

person1 = {
  "name": "John",
  "age": 36,
  "country": "Norway"
}

مثال

فقط دیکشنری person1 را از ماژول import کنید:

from mymodule import person1

print(person1["age"])

توجه: هنگام import کردن با استفاده از کلیدواژه from، از نام ماژول هنگام اشاره به عناصر در ماژول استفاده نکنید. مثال: person1["age"]، نه mymodule.person1["age"]

آزمون ماژول‌ها

برای اطمینان از درک مطالب، لطفاً به سوالات زیر پاسخ دهید:

۱. برای ایجاد یک ماژول در پایتون، فایل باید چه پسوندی داشته باشد؟
۲. کدام دستور برای import کردن یک ماژول به نام "mymodule" صحیح است؟
۳. چگونه می‌توانید هنگام import کردن یک ماژول، نام مستعار برای آن ایجاد کنید؟
