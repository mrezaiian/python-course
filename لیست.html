لیست‌ها در پایتون

mylist = ["apple", "banana", "cherry"]

لیست چیست؟

لیست‌ها برای ذخیره چندین آیتم در یک متغیر استفاده می‌شوند.

لیست‌ها یکی از 4 نوع داده داخلی در پایتون برای ذخیره مجموعه‌ای از داده‌ها هستند. سه نوع دیگر Tuple، Set و Dictionary هستند که هر کدام ویژگی‌ها و کاربردهای متفاوتی دارند.

لیست‌ها با استفاده از براکت‌های مربعی ایجاد می‌شوند:

thislist = ["apple", "banana", "cherry"] print(thislist)

خروجی:

['apple', 'banana', 'cherry']

ویژگی‌های آیتم‌های لیست

آیتم‌های لیست مرتب شده، قابل تغییر و اجازه مقادیر تکراری را می‌دهند.

آیتم‌های لیست ایندکس‌دار هستند، اولین آیتم دارای ایندکس [0]، دومین آیتم دارای ایندکس [1] و غیره است.

مرتب شده

وقتی می‌گوییم لیست‌ها مرتب شده‌اند، به این معنی است که آیتم‌ها دارای ترتیب مشخصی هستند و آن ترتیب تغییر نخواهد کرد.

اگر آیتم‌های جدیدی به لیست اضافه کنید، آیتم‌های جدید در انتهای لیست قرار می‌گیرند.

توجه: برخی از متدهای لیست ترتیب را تغییر می‌دهند، اما به طور کلی: ترتیب آیتم‌ها تغییر نخواهد کرد.

قابل تغییر

لیست قابل تغییر است، به این معنی که می‌توانیم آیتم‌های لیست را پس از ایجاد آن تغییر، اضافه و حذف کنیم.

اجازه مقادیر تکراری

از آنجایی که لیست‌ها ایندکس‌دار هستند، لیست‌ها می‌توانند آیتم‌هایی با مقدار یکسان داشته باشند:

thislist = ["apple", "banana", "cherry", "apple", "cherry"] print(thislist)

خروجی:

['apple', 'banana', 'cherry', 'apple', 'cherry']

طول لیست

برای تعیین تعداد آیتم‌های یک لیست، از تابع len() استفاده کنید:

thislist = ["apple", "banana", "cherry"] print(len(thislist))

خروجی:

3

انواع داده‌ای آیتم‌های لیست

آیتم‌های لیست می‌توانند از هر نوع داده‌ای باشند:

list1 = ["apple", "banana", "cherry"] # رشته list2 = [1, 5, 7, 9, 3] # عدد list3 = [True, False, False] # بولین

یک لیست می‌تواند شامل انواع داده‌های مختلف باشد:

list1 = ["abc", 34, True, 40, "male"]

تابع type()

از دیدگاه پایتون، لیست‌ها به عنوان اشیاء با نوع داده 'list' تعریف می‌شوند:

mylist = ["apple", "banana", "cherry"] print(type(mylist))

خروجی:

<class 'list'>

سازنده list()

همچنین می‌توان از سازنده list() برای ایجاد یک لیست جدید استفاده کرد:

thislist = list(("apple", "banana", "cherry")) # توجه به پرانتزهای دوتایی print(thislist)

خروجی:

['apple', 'banana', 'cherry']

دسترسی به آیتم‌های لیست

دسترسی با ایندکس

آیتم‌های لیست ایندکس‌دار هستند و می‌توانید با اشاره به شماره ایندکس به آن‌ها دسترسی پیدا کنید:

thislist = ["apple", "banana", "cherry"] print(thislist[1])

خروجی:

banana
توجه: اولین آیتم دارای ایندکس 0 است.

ایندکس‌گذاری منفی

ایندکس‌گذاری منفی به معنای شروع از انتها است:

thislist = ["apple", "banana", "cherry"] print(thislist[-1])

خروجی:

cherry

محدوده ایندکس‌ها

می‌توانید با مشخص کردن نقطه شروع و پایان، یک محدوده از ایندکس‌ها را مشخص کنید:

thislist = ["apple", "banana", "cherry", "orange", "kiwi", "melon", "mango"] print(thislist[2:5])

خروجی:

['cherry', 'orange', 'kiwi']
توجه: جستجو از ایندکس 2 (شامل) شروع می‌شود و در ایندکس 5 (شامل نیست) پایان می‌یابد.

تغییر آیتم‌های لیست

تغییر مقدار آیتم

برای تغییر مقدار یک آیتم خاص، به شماره ایندکس آن اشاره کنید:

thislist = ["apple", "banana", "cherry"] thislist[1] = "blackcurrant" print(thislist)

خروجی:

['apple', 'blackcurrant', 'cherry']

تغییر یک محدوده از مقادیر آیتم

برای تغییر مقادیر آیتم‌ها در یک محدوده خاص، یک لیست با مقادیر جدید تعریف کنید و به محدوده ایندکس‌هایی که می‌خواهید مقادیر جدید را درج کنید اشاره کنید:

thislist = ["apple", "banana", "cherry", "orange", "kiwi", "mango"] thislist[1:3] = ["blackcurrant", "watermelon"] print(thislist)

خروجی:

['apple', 'blackcurrant', 'watermelon', 'orange', 'kiwi', 'mango']

اضافه کردن آیتم‌های لیست

افزودن با append()

برای افزودن یک آیتم به انتهای لیست، از متد append() استفاده کنید:

thislist = ["apple", "banana", "cherry"] thislist.append("orange") print(thislist)

خروجی:

['apple', 'banana', 'cherry', 'orange']

درج با insert()

برای درج یک آیتم لیست در یک ایندکس مشخص، از متد insert() استفاده کنید:

thislist = ["apple", "banana", "cherry"] thislist.insert(1, "orange") print(thislist)

خروجی:

['apple', 'orange', 'banana', 'cherry']

افزودن با extend()

برای افزودن عناصر از یک لیست دیگر به لیست فعلی، از متد extend() استفاده کنید:

thislist = ["apple", "banana", "cherry"] tropical = ["mango", "pineapple", "papaya"] thislist.extend(tropical) print(thislist)

خروجی:

['apple', 'banana', 'cherry', 'mango', 'pineapple', 'papaya']

حذف آیتم‌های لیست

حذف با remove()

متد remove() آیتم مشخص شده را حذف می‌کند:

thislist = ["apple", "banana", "cherry"] thislist.remove("banana") print(thislist)

خروجی:

['apple', 'cherry']

حذف با pop()

متد pop() ایندکس مشخص شده را حذف می‌کند:

thislist = ["apple", "banana", "cherry"] thislist.pop(1) print(thislist)

خروجی:

['apple', 'cherry']

حذف با del

کلمه کلیدی del نیز ایندکس مشخص شده را حذف می‌کند:

thislist = ["apple", "banana", "cherry"] del thislist[0] print(thislist)

خروجی:

['banana', 'cherry']

حلقه‌ها در لیست‌ها

حلقه for

می‌توانید با استفاده از حلقه for در آیتم‌های لیست حلقه بزنید:

thislist = ["apple", "banana", "cherry"] for x in thislist: print(x)

خروجی:

apple banana cherry

درک لیست (List Comprehension)

درک لیست سینتکس کوتاه‌تری را ارائه می‌دهد وقتی می‌خواهید یک لیست جدید بر اساس مقادیر یک لیست موجود ایجاد کنید:

fruits = ["apple", "banana", "cherry", "kiwi", "mango"] newlist = [x for x in fruits if "a" in x] print(newlist)

خروجی:

['apple', 'banana', 'mango']

مرتب‌سازی لیست‌ها

مرتب‌سازی با sort()

اشیاء لیست یک متد sort() دارند که لیست را به طور پیش‌فرض به صورت صعودی مرتب می‌کند:

thislist = ["orange", "mango", "kiwi", "pineapple", "banana"] thislist.sort() print(thislist)

خروجی:

['banana', 'kiwi', 'mango', 'orange', 'pineapple']

مرتب‌سازی نزولی

برای مرتب‌سازی نزولی، از آرگومان کلیدی reverse = True استفاده کنید:

thislist = ["orange", "mango", "kiwi", "pineapple", "banana"] thislist.sort(reverse = True) print(thislist)

خروجی:

['pineapple', 'orange', 'mango', 'kiwi', 'banana']

کپی لیست‌ها

شما نمی‌توانید یک لیست را فقط با تایپ کردن list2 = list1 کپی کنید، زیرا list2 فقط یک ارجاع به list1 خواهد بود و تغییرات ایجاد شده در list1 به طور خودکار در list2 نیز انجام می‌شود.

استفاده از متد copy()

thislist = ["apple", "banana", "cherry"] mylist = thislist.copy() print(mylist)

خروجی:

['apple', 'banana', 'cherry']

اتصال لیست‌ها

اتصال با عملگر +

list1 = ["a", "b", "c"] list2 = [1, 2, 3] list3 = list1 + list2 print(list3)

خروجی:

['a', 'b', 'c', 1, 2, 3]

متدهای لیست

متد توضیح
append() یک عنصر را به انتهای لیست اضافه می‌کند
clear() همه عناصر را از لیست حذف می‌کند
copy() یک کپی از لیست برمی‌گرداند
count() تعداد عناصر با مقدار مشخص شده را برمی‌گرداند
extend() عناصر یک لیست (یا هر تکرارپذیر) را به انتهای لیست فعلی اضافه می‌کند
index() ایندکس اولین عنصر با مقدار مشخص شده را برمی‌گرداند
insert() یک عنصر را در موقعیت مشخص شده اضافه می‌کند
pop() عنصر را در موقعیت مشخص شده حذف می‌کند
remove() آیتم را با مقدار مشخص شده حذف می‌کند
reverse() ترتیب لیست را معکوس می‌کند
sort() لیست را مرتب می‌کند
