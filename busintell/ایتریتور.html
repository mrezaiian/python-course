Python Iterators

ایتریتور چیست؟

ایتریتور یک شیء است که شامل تعداد قابل شمارشی از مقادیر است.

ایتریتور یک شیء است که می‌توان روی آن تکرار کرد، به این معنی که می‌توانید از میان تمام مقادیر آن پیمایش کنید.

از نظر فنی، در پایتون، یک ایتریتور یک شیء است که پروتکل ایتریتور را پیاده‌سازی می‌کند که شامل متدهای __iter__() و __next__() است.

ایتریتور در مقابل ایتربل

لیست‌ها، تاپل‌ها، دیکشنری‌ها و مجموعه‌ها همه اشیاء قابل تکرار هستند. آن‌ها کانتینرهای قابل تکرار هستند که می‌توانید از آن‌ها یک ایتریتور دریافت کنید.

همه این اشیاء یک متد iter() دارند که برای دریافت یک ایتریتور استفاده می‌شود:

مثال

یک ایتریتور از یک تاپل برگردانید و هر مقدار را چاپ کنید:

mytuple = ("apple", "banana", "cherry")
myit = iter(mytuple)

print(next(myit))
print(next(myit))
print(next(myit))

حتی رشته‌ها نیز اشیاء قابل تکرار هستند و می‌توانند یک ایتریتور برگردانند:

مثال

رشته‌ها نیز اشیاء قابل تکرار هستند که شامل دنباله‌ای از کاراکترها هستند:

mystr = "banana"
myit = iter(mystr)

print(next(myit))
print(next(myit))
print(next(myit))
print(next(myit))
print(next(myit))
print(next(myit))

تست اول: درک مفاهیم پایه

کدام یک از موارد زیر در پایتون قابل تکرار (iterable) نیست؟

حلقه زدن در ایتریتور

ما همچنین می‌توانیم از یک حلقه for برای تکرار در یک شیء قابل تکرار استفاده کنیم:

مثال

مقادیر یک تاپل را تکرار کنید:

mytuple = ("apple", "banana", "cherry")

for x in mytuple:
    print(x)

مثال

کاراکترهای یک رشته را تکرار کنید:

mystr = "banana"

for x in mystr:
    print(x)

حلقه for در واقع یک شیء ایتریتور ایجاد می‌کند و متد next() را برای هر حلقه اجرا می‌کند.

تست دوم: حلقه‌ها و ایتریتورها

کدام یک از گزینه‌های زیر در مورد حلقه for در پایتون صحیح است؟

ایجاد یک ایتریتور

برای ایجاد یک شیء/کلاس به عنوان ایتریتور، باید متدهای __iter__() و __next__() را برای شیء خود پیاده‌سازی کنید.

همانطور که در فصل کلاس‌ها/اشیاء پایتون یاد خواهید گرفت، همه کلاس‌ها یک تابع به نام __init__() دارند که به شما امکان می‌دهد هنگام ایجاد شیء، مقداری مقداردهی اولیه انجام دهید.

متد __iter__() به طور مشابه عمل می‌کند، شما می‌توانید عملیات (مقداردهی اولیه و غیره) انجام دهید، اما باید همیشه خود شیء ایتریتور را برگرداند.

متد __next__() نیز به شما امکان انجام عملیات را می‌دهد و باید مورد بعدی در دنباله را برگرداند.

مثال

یک ایتریتور ایجاد کنید که اعداد را برگرداند، از 1 شروع می‌شود و هر دنباله یک عدد افزایش می‌یابد (1،2،3،4،5 و غیره را برمی‌گرداند):

class MyNumbers:
    def __iter__(self):
        self.a = 1
        return self

    def __next__(self):
        x = self.a
        self.a += 1
        return x

myclass = MyNumbers()
myiter = iter(myclass)

print(next(myiter))
print(next(myiter))
print(next(myiter))
print(next(myiter))
print(next(myiter))

StopIteration

مثال بالا اگر به اندازه کافی عبارت next() داشته باشید یا اگر در یک حلقه for استفاده شود، برای همیشه ادامه خواهد یافت.

برای جلوگیری از ادامه یافتن تکرار برای همیشه، می‌توانیم از عبارت StopIteration استفاده کنیم.

در متد __next__()، می‌توانیم یک شرط خاتمه اضافه کنیم تا در صورت انجام تکرار به تعداد مشخص شده، یک خطا ایجاد کند:

مثال

بعد از 20 تکرار متوقف شود:

class MyNumbers:
    def __iter__(self):
        self.a = 1
        return self

    def __next__(self):
        if self.a <= 20:
            x = self.a
            self.a += 1
            return x
        else:
            raise StopIteration

myclass = MyNumbers()
myiter = iter(myclass)

for x in myiter:
    print(x)
