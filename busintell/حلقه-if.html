حلقه if در پایتون

شرایط و دستورات if در پایتون

پایتون از شرایط منطقی معمول در ریاضیات پشتیبانی می‌کند:

  • مساوی: a == b
  • نا مساوی: a != b
  • کمتر از: a < b
  • کمتر یا مساوی: a <= b
  • بیشتر از: a > b
  • بیشتر یا مساوی: a >= b

این شرایط می‌توانند به چندین روش استفاده شوند، که معمولاً در "دستورات if" و حلقه‌ها به کار می‌روند.

یک "دستور if" با استفاده از کلمه کلیدی if نوشته می‌شود.

مثال: دستور if

a = 33
b = 200
if b > a:
  print("b is greater than a")

در این مثال از دو متغیر a و b استفاده کرده‌ایم که به عنوان بخشی از دستور if برای آزمایش اینکه آیا b از a بزرگتر است استفاده می‌شوند. از آنجایی که a برابر با 33 و b برابر با 200 است، می‌دانیم که 200 از 33 بزرگتر است، بنابراین روی صفحه چاپ می‌کنیم که "b بزرگتر از a است".

چگونه دستورات if کار می‌کنند

دستور if یک شرط (یک عبارت که نتیجه True یا False دارد) را ارزیابی می‌کند. اگر شرط درست باشد، بلوک کد داخل دستور if اجرا می‌شود. اگر شرط نادرست باشد، بلوک کد رد می‌شود.

مثال: بررسی مثبت بودن یک عدد

number = 15
if number > 0:
  print("The number is positive")

تورفتگی (Indentation)

پایتون برای تعریف محدوده در کد به تورفتگی (فضای خالی در ابتدای خط) متکی است. زبان‌های برنامه‌نویسی دیگر اغلب از آکولاد برای این منظور استفاده می‌کنند.

مثال: دستور if بدون تورفتگی (خطا ایجاد می‌کند)

a = 33
b = 200
if b > a:
print("b is greater than a") # شما یک خطا دریافت خواهید کرد

توجه: شما می‌توانید از فضاها یا تب‌ها برای تورفتگی استفاده کنید، اما باید از همان مقدار تورفتگی برای تمام دستورات درون یک بلوک کد استفاده کنید.

آزمون: درک دستور if

خروجی کد زیر چیست؟

x = 10
y = 5
if x > y:
  print("x is greater")

دستور Elif در پایتون

کلمه کلیدی Elif

کلمه کلیدی elif روش پایتون برای گفتن "اگر شرایط قبلی درست نبود، این شرط را امتحان کن" است.

کلمه کلیدی elif به شما امکان می‌دهد چندین عبارت را برای True بررسی کنید و به محض ارزیابی یکی از شرایط به True، یک بلوک کد اجرا کنید.

مثال: دستور elif

a = 33
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")

در این مثال a برابر با b است، بنابراین شرط اول درست نیست، اما شرط elif درست است، بنابراین روی صفحه چاپ می‌کنیم که "a و b برابر هستند".

آزمون: درک دستور elif

خروجی کد زیر چیست؟

score = 75
if score >= 90:
  print("Grade: A")
elif score >= 80:
  print("Grade: B")
elif score >= 70:
  print("Grade: C")
elif score >= 60:
  print("Grade: D")

دستور Else در پایتون

کلمه کلیدی Else

کلمه کلیدی else هر چیزی را که توسط شرایط قبلی گرفته نشده است، می‌گیرد.

دستور else زمانی اجرا می‌شود که شرط if (و هر شرط elif) به False ارزیابی شود.

مثال: دستور else

a = 200
b = 33
if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")
else:
  print("a is greater than b")

در این مثال a از b بزرگتر است، بنابراین شرط اول درست نیست، همچنین شرط elif نیز درست نیست، بنابراین به شرط else می‌رویم و روی صفحه چاپ می‌کنیم که "a بزرگتر از b است".

آزمون: درک دستور else

خروجی کد زیر چیست؟

number = 7
if number % 2 == 0:
  print("The number is even")
else:
  print("The number is odd")

عملگرهای منطقی در پایتون

عملگرهای منطقی برای ترکیب دستورات شرطی استفاده می‌شوند. پایتون سه عملگر منطقی دارد:

  • and - اگر هر دو عبارت درست باشند، True برمی‌گرداند
  • or - اگر یکی از عبارات درست باشد، True برمی‌گرداند
  • not - نتیجه را معکوس می‌کند، اگر نتیجه درست باشد False برمی‌گرداند

مثال: عملگر and

a = 200
b = 33
c = 500
if a > b and c > a:
  print("Both conditions are True")

مثال: عملگر or

a = 200
b = 33
c = 500
if a > b or a > c:
  print("At least one of the conditions is True")

آزمون: درک عملگرهای منطقی

خروجی کد زیر چیست؟

x = 10
y = 20
z = 30
if x < y and y < z:
  print("All conditions are true")

دستورات if تودرتو

شما می‌توانید دستورات if را درون دستورات if داشته باشید. این را دستورات if تودرتو می‌نامند.

مثال: دستورات if تودرتو

x = 41
if x > 10:
  print("Above ten,")
  if x > 20:
    print("and also above 20!")
  else:
    print("but not above 20.")

در این مثال، دستور if داخلی فقط در صورتی اجرا می‌شود که شرط بیرونی (x > 10) درست باشد.

آزمون: درک if تودرتو

خروجی کد زیر چیست؟

age = 25
has_license = True
if age >= 18:
  if has_license:
    print("You can drive")
  else:
    print("You need a license")
else:
  print("You are too young to drive")

دستور Pass در پایتون

دستور pass

دستورات if نمی‌توانند خالی باشند، اما اگر به هر دلیلی یک دستور if بدون محتوا دارید، دستور pass را قرار دهید تا از دریافت خطا جلوگیری کنید.

مثال: دستور pass

a = 33
b = 200
if b > a:
  pass

دستور pass یک عملیات null است - وقتی اجرا می‌شود هیچ اتفاقی نمی‌افتد. این به عنوان یک نگهدارنده عمل می‌کند.

چرا از pass استفاده کنیم؟ دستور pass در چندین موقعیت مفید است:

  • وقتی در حال ایجاد ساختار کد هستید اما هنوز منطق را پیاده‌سازی نکرده‌اید
  • وقتی یک دستور از نظر نحوی لازم است اما هیچ عملی لازم نیست
  • به عنوان نگهدارنده برای کد آینده در طول توسعه
