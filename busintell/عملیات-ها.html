عملیات‌ها در پایتون

عملگرهای حسابی پایتون

عملگرهای حسابی برای انجام عملیات ریاضی روی مقادیر عددی استفاده می‌شوند:

عملگر نام مثال
+ جمع x + y
- تفریق x - y
* ضرب x * y
/ تقسیم x / y
% باقیمانده x % y
** توان x ** y
// تقسیم صحیح x // y

مثال‌ها

در اینجا یک مثال با استفاده از عملگرهای حسابی مختلف آورده شده است:

x = 15 y = 4 print(x + y) # 19 print(x - y) # 11 print(x * y) # 60 print(x / y) # 3.75 print(x % y) # 3 print(x ** y) # 50625 print(x // y) # 3

آزمون: عملگرهای حسابی

خروجی کد زیر چیست؟

x = 10 y = 3 print(x // y)

تقسیم در پایتون

پایتون دو عملگر تقسیم دارد:

  • / - تقسیم (یک عدد اعشاری برمی‌گرداند)
  • // - تقسیم صحیح (یک عدد صحیح برمی‌گرداند)

مثال

تقسیم همیشه یک عدد اعشاری برمی‌گرداند:

x = 12 y = 5 print(x / y) # 2.4

مثال

تقسیم صحیح همیشه یک عدد صحیح برمی‌گرداند. این عملگر به سمت پایین گرد می‌کند:

x = 12 y = 5 print(x // y) # 2

درباره BusIntell

معرفی پلتفرم

BusIntell یک پلتفرم آموزشی آنلاین است که با هدف ارائه آموزش‌های باکیفیت برنامه‌نویسی پایتون به زبان فارسی ایجاد شده است. ما بر یادگیری تعاملی و پروژه‌محور تمرکز داریم.

فلسفه آموزشی

ما معتقدیم بهترین روش یادگیری برنامه‌نویسی، تمرین عملی و حل مسئله است. به همین دلیل، در کنار هر درس، تمرین‌های تعاملی و پروژه‌های واقعی ارائه می‌دهیم.

تیم ما

تیم BusIntell متشکل از توسعه‌دهندگان و مربیان با تجربه است که علاقه‌مند به اشتراک‌گذاری دانش و کمک به رشد جامعه برنامه‌نویسی فارسی‌زبان هستند.
