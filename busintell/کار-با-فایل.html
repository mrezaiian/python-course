کار با فایل در پایتون

مدیریت فایل بخش مهمی از هر برنامه کاربردی وب است. پایتون چندین تابع برای ایجاد، خواندن، به‌روزرسانی و حذف فایل‌ها دارد.

مدیریت فایل

تابع کلیدی برای کار با فایل‌ها در پایتون، تابع open() است.

تابع open() دو پارامتر می‌گیرد: نام فایل و حالت.

چهار روش (حالت) مختلف برای باز کردن فایل وجود دارد:

  • "r" - خواندن - مقدار پیش‌فرض. فایل را برای خواندن باز می‌کند، اگر فایل وجود نداشته باشد خطا می‌دهد
  • "a" - الحاق - فایل را برای الحاق باز می‌کند، اگر فایل وجود نداشته باشد آن را ایجاد می‌کند
  • "w" - نوشتن - فایل را برای نوشتن باز می‌کند، اگر فایل وجود نداشته باشد آن را ایجاد می‌کند
  • "x" - ایجاد - فایل مشخص شده را ایجاد می‌کند، اگر فایل وجود داشته باشد خطا می‌دهد

علاوه بر این، می‌توانید مشخص کنید که فایل باید به صورت باینری یا متنی مدیریت شود:

  • "t" - متن - مقدار پیش‌فرض. حالت متن
  • "b" - باینری - حالت باینری (مثلاً برای تصاویر)

سینتکس

برای باز کردن فایل برای خواندن، کافی است نام فایل را مشخص کنید:

f = open("demofile.txt")

کد بالا معادل است با:

f = open("demofile.txt", "rt")
توجه: مطمئن شوید فایل وجود دارد، در غیر این صورت خطا دریافت خواهید کرد.

باز کردن فایل در سرور

فرض کنید فایل زیر را داریم که در همان پوشه پایتون قرار دارد:

demofile.txt

Hello! Welcome to demofile.txt
This file is for testing purposes.
Good Luck!

برای باز کردن فایل، از تابع داخلی open() استفاده کنید.

تابع open() یک شیء فایل برمی‌گرداند که یک متد read() برای خواندن محتوای فایل دارد:

مثال

f = open("demofile.txt")
print(f.read())

اگر فایل در مکان دیگری قرار دارد، باید مسیر فایل را مشخص کنید:

مثال

باز کردن فایل در مکان متفاوت:

f = open("D:\\myfiles\\welcome.txt")
print(f.read())

استفاده از عبارت with

همچنین می‌توانید از عبارت with هنگام باز کردن فایل استفاده کنید:

مثال

استفاده از کلیدواژه with:

with open("demofile.txt") as f:
  print(f.read())

در این صورت نیازی به نگرانی درباره بستن فایل‌های خود ندارید، عبارت with این کار را انجام می‌دهد.

بستن فایل‌ها

همیشه بستن فایل پس از اتمام کار با آن یک عمل خوب است.

اگر از عبارت with استفاده نمی‌کنید، باید یک عبارت close برای بستن فایل بنویسید:

مثال

بستن فایل پس از اتمام کار با آن:

f = open("demofile.txt")
print(f.readline())
f.close()
توجه: همیشه باید فایل‌های خود را ببندید. در برخی موارد، به دلیل بافرینگ، تغییرات اعمال شده روی یک فایل ممکن است تا زمانی که فایل را نبندید، نمایش داده نشوند.

خواندن فقط بخشی از فایل

به طور پیش‌فرض، متد read() کل متن را برمی‌گرداند، اما می‌توانید مشخص کنید که چند کاراکتر می‌خواهید برگردانید:

مثال

برگرداندن 5 کاراکتر اول فایل:

with open("demofile.txt") as f:
  print(f.read(5))

خواندن خطوط

می‌توانید یک خط را با استفاده از متد readline() برگردانید:

مثال

خواندن یک خط از فایل:

with open("demofile.txt") as f:
  print(f.readline())

با دو بار فراخوانی readline()، می‌توانید دو خط اول را بخوانید:

مثال

خواندن دو خط از فایل:

with open("demofile.txt") as f:
  print(f.readline())
  print(f.readline())

با حلقه زدن در خطوط فایل، می‌توانید کل فایل را خط به خط بخوانید:

مثال

حلقه زدن در فایل خط به خط:

with open("demofile.txt") as f:
  for x in f:
    print(x)

نوشتن در فایل پایتون

نوشتن در فایل موجود

برای نوشتن در فایل موجود، باید یک پارامتر به تابع open() اضافه کنید:

  • "a" - الحاق - به انتهای فایل اضافه می‌کند
  • "w" - نوشتن - هر محتوای موجود را بازنویسی می‌کند

مثال

باز کردن فایل "demofile.txt" و اضافه کردن محتوا به فایل:

with open("demofile.txt", "a") as f:
  f.write("Now the file has more content!")

# باز کردن و خواندن فایل پس از الحاق:
with open("demofile.txt") as f:
  print(f.read())

بازنویسی محتوای موجود

برای بازنویسی محتوای موجود در فایل، از پارامتر w استفاده کنید:

مثال

باز کردن فایل "demofile.txt" و بازنویسی محتوا:

with open("demofile.txt", "w") as f:
  f.write("Woops! I have deleted the content!")

# باز کردن و خواندن فایل پس از بازنویسی:
with open("demofile.txt") as f:
  print(f.read())
توجه: روش "w" کل فایل را بازنویسی می‌کند.

ایجاد فایل جدید

برای ایجاد فایل جدید در پایتون، از متد open() با یکی از پارامترهای زیر استفاده کنید:

  • "x" - ایجاد - یک فایل ایجاد می‌کند، اگر فایل وجود داشته باشد خطا برمی‌گرداند
  • "a" - الحاق - اگر فایل مشخص شده وجود نداشته باشد، فایل را ایجاد می‌کند
  • "w" - نوشتن - اگر فایل مشخص شده وجود نداشته باشد، فایل را ایجاد می‌کند

مثال

ایجاد فایل جدید به نام "myfile.txt":

f = open("myfile.txt", "x")

نتیجه: یک فایل خالی جدید ایجاد می‌شود.

توجه: اگر فایل از قبل وجود داشته باشد، خطا ایجاد می‌شود.

حذف فایل پایتون

حذف فایل

برای حذف فایل، باید ماژول OS را وارد کنید و تابع os.remove() آن را اجرا کنید:

مثال

حذف فایل "demofile.txt":

import os
os.remove("demofile.txt")

بررسی وجود فایل:

برای جلوگیری از دریافت خطا، ممکن است بخواهید قبل از تلاش برای حذف فایل، بررسی کنید که فایل وجود دارد:

مثال

بررسی وجود فایل، سپس حذف آن:

import os
if os.path.exists("demofile.txt"):
  os.remove("demofile.txt")
else:
  print("The file does not exist")

حذف پوشه

برای حذف کل یک پوشه، از متد os.rmdir() استفاده کنید:

مثال

حذف پوشه "myfolder":

import os
os.rmdir("myfolder")
توجه: شما فقط می‌توانید پوشه‌های خالی را حذف کنید.
