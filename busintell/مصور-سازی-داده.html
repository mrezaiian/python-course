مصورسازی داده با Python

مصورسازی داده‌ها یکی از مهارت‌های مهم در علم داده است که به ما کمک می‌کند تا الگوها، روندها و روابط موجود در داده‌ها را بهتر درک کنیم. در این بخش با استفاده از کتابخانه matplotlib در پایتون، نحوه ایجاد نمودارهای مختلف را یاد خواهیم گرفت.

معرفی matplotlib

matplotlib یک کتابخانه قدرتمند پایتون برای ایجاد نمودارها و مصورسازی داده‌ها است. این کتابخانه امکان ایجاد انواع نمودارها از جمله خطی، میله‌ای، دایره‌ای، هیستوگرام و... را فراهم می‌کند.

# نصب و وارد کردن کتابخانه
pip install matplotlib
import matplotlib.pyplot as plt
import numpy as np

ایجاد نمودار پایه

برای ایجاد یک نمودار ساده، ابتدا داده‌های خود را تعریف کرده و سپس با استفاده از تابع plot آن‌ها را رسم می‌کنیم.

# ایجاد داده‌های نمونه
x = [1, 2, 3, 4, 5]
y = [2, 4, 6, 8, 10]

# رسم نمودار خطی
plt.plot(x, y)
plt.title('نمودار خطی نمونه')
plt.xlabel('محور X')
plt.ylabel('محور Y')
plt.show()
