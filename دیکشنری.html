دیکشنری پایتون

دیکشنری چیست؟

دیکشنری‌ها برای ذخیره مقادیر داده در قالب جفت‌های کلید:مقدار استفاده می‌شوند.

یک دیکشنری مجموعه‌ای است که مرتب‌شده، قابل تغییر است و اجازه تکراری‌ها را نمی‌دهد.

از نسخه 3.7 پایتون، دیکشنری‌ها مرتب‌شده هستند. در پایتون 3.6 و نسخه‌های قبل، دیکشنری‌ها نامرتب بودند.

دیکشنری‌ها با آکولاد نوشته می‌شوند و دارای کلید و مقدار هستند:

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
print(thisdict)

آیتم‌های دیکشنری

آیتم‌های دیکشنری مرتب‌شده، قابل تغییر هستند و اجازه تکراری‌ها را نمی‌دهند.

آیتم‌های دیکشنری به صورت جفت‌های کلید:مقدار ارائه می‌شوند و می‌توان با استفاده از نام کلید به آن‌ها ارجاع داد.

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
print(thisdict["brand"])

مرتب‌شده یا نامرتب؟

از نسخه 3.7 پایتون، دیکشنری‌ها مرتب‌شده هستند. در پایتون 3.6 و نسخه‌های قبل، دیکشنری‌ها نامرتب بودند.

وقتی می‌گوییم دیکشنری‌ها مرتب‌شده هستند، یعنی آیتم‌ها دارای ترتیب مشخصی هستند و این ترتیب تغییر نخواهد کرد.

نامرتب بودن به این معنی است که آیتم‌ها ترتیب مشخصی ندارند و نمی‌توان با استفاده از ایندکس به یک آیتم ارجاع داد.

قابل تغییر

دیکشنری‌ها قابل تغییر هستند، به این معنی که می‌توانیم پس از ایجاد دیکشنری، آیتم‌ها را تغییر داده، اضافه یا حذف کنیم.

عدم اجازه تکراری‌ها

دیکشنری‌ها نمی‌توانند دو آیتم با کلید یکسان داشته باشند:

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964,
  "year": 2020
}
print(thisdict)

مقادیر تکراری، مقادیر موجود را بازنویسی می‌کنند.

طول دیکشنری

برای تعیین تعداد آیتم‌های یک دیکشنری، از تابع len() استفاده کنید:

print(len(thisdict))

آیتم‌های دیکشنری - انواع داده

مقادیر در آیتم‌های دیکشنری می‌توانند از هر نوع داده‌ای باشند:

thisdict = {
  "brand": "Ford",
  "electric": False,
  "year": 1964,
  "colors": ["red", "white", "blue"]
}

type()

از دیدگاه پایتون، دیکشنری‌ها به عنوان اشیاء با نوع داده 'dict' تعریف می‌شوند:

thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
print(type(thisdict))

سازنده dict()

همچنین می‌توان از سازنده dict() برای ایجاد یک دیکشنری استفاده کرد:

thisdict = dict(name = "John", age = 36, country = "Norway")
print(thisdict)

مجموعه‌های پایتون (آرایه‌ها)

چهار نوع داده مجموعه در زبان برنامه‌نویسی پایتون وجود دارد:

  • لیست: مجموعه‌ای است که مرتب‌شده و قابل تغییر است. اجازه اعضای تکراری را می‌دهد.
  • تاپل: مجموعه‌ای است که مرتب‌شده و غیرقابل تغییر است. اجازه اعضای تکراری را می‌دهد.
  • مجموعه: مجموعه‌ای است که نامرتب، غیرقابل تغییر* و بدون ایندکس است. هیچ عضو تکراری ندارد.
  • دیکشنری: مجموعه‌ای است که مرتب‌شده** و قابل تغییر است. هیچ عضو تکراری ندارد.

*آیتم‌های مجموعه غیرقابل تغییر هستند، اما می‌توانید هر زمان که بخواهید آیتم‌ها را حذف و/یا اضافه کنید.

**از نسخه 3.7 پایتون، دیکشنری‌ها مرتب‌شده هستند. در پایتون 3.6 و نسخه‌های قبل، دیکشنری‌ها نامرتب بودند.
