متغیرها در پایتون

متغیرها

متغیرها ظروفی برای ذخیره مقادیر داده هستند.

ایجاد متغیرها

پایتون هیچ دستوری برای اعلام متغیر ندارد. یک متغیر در لحظه‌ای که اولین بار به آن مقدار اختصاص می‌دهید، ایجاد می‌شود.

مثال
x = 5
y = "John"
print(x)
print(y)

متغیرها نیازی به اعلام با نوع خاصی ندارند و حتی می‌توانند پس از تنظیم، نوع خود را تغییر دهند.

مثال
x = 4 # x از نوع int است
x = "Sally" # x اکنون از نوع str است
print(x)

Casting

اگر می‌خواهید نوع داده یک متغیر را مشخص کنید، این کار را می‌توان با casting انجام داد.

مثال
x = str(3) # x برابر '3' خواهد بود
y = int(3) # y برابر 3 خواهد بود
z = float(3) # z برابر 3.0 خواهد بود

دریافت نوع

می‌توانید نوع داده یک متغیر را با تابع type() دریافت کنید.

مثال
x = 5
y = "John"
print(type(x))
print(type(y))

نقل‌قول تکی یا دوتایی؟

متغیرهای رشته‌ای را می‌توان با استفاده از نقل‌قول تکی یا دوتایی اعلام کرد:

مثال
x = "John"
# همان است
x = 'John'

حساس به حروف بزرگ و کوچک

نام متغیرها به حروف بزرگ و کوچک حساس هستند.

a = 4

برای اطمینان از درک مطلب، لطفاً به سوال زیر پاسخ دهید:

کدام یک از نام‌های متغیر زیر در پایتون معتبر است؟

نام‌های متغیر

یک متغیر می‌تواند نام کوتاه (مانند x و y) یا نام توصیفی (مانند age، carname، total_volume) داشته باشد.

قوانین برای متغیرهای پایتون:

  • نام متغیر باید با یک حرف یا کاراکتر زیرخط شروع شود
  • نام متغیر نمی‌تواند با یک عدد شروع شود
  • نام متغیر فقط می‌تواند شامل کاراکترهای آلفانومریک و زیرخط باشد (A-z، 0-9 و _)
  • نام متغیرها به حروف بزرگ و کوچک حساس هستند (age، Age و AGE سه متغیر مختلف هستند)
  • نام متغیر نمی‌تواند هیچ یک از کلمات کلیدی پایتون باشد.
نام‌های متغیر قانونی:
myvar = "John"
my_var = "John"
_my_var = "John"
myVar = "John"
MYVAR = "John"
myvar2 = "John"
نام‌های متغیر غیرقانونی:
2myvar = "John"
my-var = "John"
my var = "John"
توجه:

به یاد داشته باشید که نام متغیرها به حروف بزرگ و کوچک حساس هستند.

نام‌های متغیر چند کلمه‌ای

نام‌های متغیر با بیش از یک کلمه می‌توانند خواندن را دشوار کنند. چندین تکنیک وجود دارد که می‌توانید برای خوانایی بیشتر از آنها استفاده کنید:

Camel Case

هر کلمه، به جز اولین کلمه، با یک حرف بزرگ شروع می‌شود:

myVariableName = "John"

Pascal Case

هر کلمه با یک حرف بزرگ شروع می‌شود:

MyVariableName = "John"

Snake Case

هر کلمه با یک کاراکتر زیرخط جدا می‌شود:

my_variable_name = "John"

آزمون: نام‌گذاری متغیرها

برای اطمینان از درک مطلب، لطفاً به سوال زیر پاسخ دهید:

کدام یک از موارد زیر یک نام متغیر معتبر در پایتون نیست؟

متغیرهای پایتون - اختصاص چندین مقدار

چند مقدار به چند متغیر

پایتون به شما امکان می‌دهد مقادیر را به چندین متغیر در یک خط اختصاص دهید:

مثال
x, y, z = "Orange", "Banana", "Cherry"
print(x)
print(y)
print(z)
توجه:

مطمئن شوید که تعداد متغیرها با تعداد مقادیر مطابقت دارد، در غیر این صورت خطا دریافت خواهید کرد.

یک مقدار به چند متغیر

و می‌توانید همان مقدار را به چندین متغیر در یک خط اختصاص دهید:

مثال
x = y = z = "Orange"
print(x)
print(y)
print(z)

آزمون: اختصاص مقادیر

برای اطمینان از درک مطلب، لطفاً به سوال زیر پاسخ دهید:

x, y, z = "Apple", "Banana", "Cherry"
print(y)
