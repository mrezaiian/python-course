اعداد در پایتون

انواع عددی در پایتون

سه نوع عددی در پایتون وجود دارد:

  • int (اعداد صحیح)
  • float (اعداد اعشاری)
  • complex (اعداد مختلط)

متغیرهای عددی زمانی ایجاد می‌شوند که شما یک مقدار را به آن‌ها اختصاص دهید:

x = 1 # int
y = 2.8 # float
z = 1j # complex

برای تأیید نوع هر شی در پایتون، از تابع type() استفاده کنید:

print(type(x))
print(type(y))
print(type(z))

اعداد صحیح (Int)

Int یا عدد صحیح، یک عدد کامل، مثبت یا منفی، بدون اعشار و با طول نامحدود است.

# اعداد صحیح:
x = 1
y = 35656222554887711
z = -3255522

print(type(x))
print(type(y))
print(type(z))

اعداد اعشاری (Float)

Float یا "عدد اعشاری" یک عدد مثبت یا منفی است که شامل یک یا چند رقم اعشار می‌باشد.

# اعداد اعشاری:
x = 1.10
y = 1.0
z = -35.59

print(type(x))
print(type(y))
print(type(z))

اعداد اعشاری می‌توانند به صورت علمی نیز نوشته شوند که با "e" توان 10 را نشان می‌دهند.

# اعداد اعشاری علمی:
x = 35e3
y = 12E4
z = -87.7e100

print(type(x))
print(type(y))
print(type(z))

اعداد مختلط (Complex)

اعداد مختلط با "j" به عنوان بخش موهومی نوشته می‌شوند:

# اعداد مختلط:
x = 3+5j
y = 5j
z = -5j

print(type(x))
print(type(y))
print(type(z))

تبدیل نوع

شما می‌توانید از یک نوع به نوع دیگر با استفاده از توابع int()، float() و complex() تبدیل کنید:

x = 1 # int
y = 2.8 # float
z = 1j # complex

# تبدیل از int به float:
a = float(x)

# تبدیل از float به int:
b = int(y)

# تبدیل از int به complex:
c = complex(x)

print(a)
print(b)
print(c)

print(type(a))
print(type(b))
print(type(c))

توجه: شما نمی‌توانید اعداد مختلط را به نوع عددی دیگری تبدیل کنید.

اعداد تصادفی

پایتون تابع random() برای تولید اعداد تصادفی ندارد، اما ماژول داخلی به نام random دارد که می‌تواند برای تولید اعداد تصادفی استفاده شود:

# وارد کردن ماژول random و نمایش یک عدد تصادفی بین 1 تا 9:
import random

print(random.randrange(1, 10))

تبدیل نوع صریح (Casting)

تبدیل نوع در پایتون با استفاده از توابع سازنده انجام می‌شود:

  • int() - یک عدد صحیح از یک عدد صحیح، عدد اعشاری (با حذف تمام اعشارها) یا رشته (در صورتی که رشته نشان‌دهنده یک عدد کامل باشد) می‌سازد
  • float() - یک عدد اعشاری از یک عدد صحیح، عدد اعشاری یا رشته (در صورتی که رشته نشان‌دهنده یک عدد اعشاری یا صحیح باشد) می‌سازد
  • str() - یک رشته از انواع مختلف داده‌ها از جمله رشته‌ها، اعداد صحیح و اعداد اعشاری می‌سازد
# اعداد صحیح:
x = int(1) # x برابر 1 خواهد بود
y = int(2.8) # y برابر 2 خواهد بود
z = int("3") # z برابر 3 خواهد بود
# اعداد اعشاری:
x = float(1) # x برابر 1.0 خواهد بود
y = float(2.8) # y برابر 2.8 خواهد بود
z = float("3") # z برابر 3.0 خواهد بود
w = float("4.2") # w برابر 4.2 خواهد بود
# رشته‌ها:
x = str("s1") # x برابر 's1' خواهد بود
y = str(2) # y برابر '2' خواهد بود
z = str(3.0) # z برابر '3.0' خواهد بود
