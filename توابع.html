توابع پایتون

یک تابع بلوکی از کد است که فقط زمانی اجرا می‌شود که فراخوانی شود.

یک تابع می‌تواند داده‌ها را به عنوان نتیجه برگرداند.

یک تابع به جلوگیری از تکرار کد کمک می‌کند.

ایجاد یک تابع

در پایتون، یک تابع با استفاده از کلمه کلیدی def تعریف می‌شود، به دنبال آن نام تابع و پرانتزها:

مثال
def greet():
  print("Hello from a function")

این یک تابع به نام greet ایجاد می‌کند که وقتی فراخوانی شود، "Hello from a function" را چاپ می‌کند.

توجه

کد داخل تابع باید تورفتگی داشته باشد. پایتون از تورفتگی برای تعریف بلوک‌های کد استفاده می‌کند.

فراخوانی یک تابع

برای فراخوانی یک تابع، نام آن را به همراه پرانتزها بنویسید:

مثال
def my_function():
  print("Hello from a function")

my_function()

شما می‌توانید همان تابع را چندین بار فراخوانی کنید:

مثال
def my_function():
  print("Hello from a function")

my_function()
my_function()
my_function()

نام‌های تابع

نام‌های تابع از همان قوانین نام‌های متغیر در پایتون پیروی می‌کنند:

  • نام تابع باید با یک حرف یا زیرخط شروع شود
  • نام تابع فقط می‌تواند شامل حروف، اعداد و زیرخط‌ها باشد
  • نام‌های تابع به حروف حساس هستند (myFunction و myfunction متفاوت هستند)
مثال
# نام‌های معتبر تابع:
calculate_sum()
_private_function()
myFunction2()

عمل خوبی است که از نام‌های توصیفی استفاده کنید که توضیح می‌دهد تابع چه کاری انجام می‌دهد.

چرا از توابع استفاده کنیم؟

تصور کنید که نیاز دارید چندین بار در برنامه خود دما را از فارنهایت به سلسیوس تبدیل کنید. بدون توابع، باید کد محاسبه یکسانی را به طور مکرر بنویسید:

مثال
# بدون توابع - کد تکراری:
temp1 = 77
celsius1 = (temp1 - 32) * 5 / 9
print(celsius1)

temp2 = 95
celsius2 = (temp2 - 32) * 5 / 9
print(celsius2)

temp3 = 50
celsius3 = (temp3 - 32) * 5 / 9
print(celsius3)

با توابع، کد را یک بار می‌نویسید و دوباره استفاده می‌کنید:

مثال
# با توابع - کد قابل استفاده مجدد:
def fahrenheit_to_celsius(fahrenheit):
  return (fahrenheit - 32) * 5 / 9

print(fahrenheit_to_celsius(77))
print(fahrenheit_to_celsius(95))
print(fahrenheit_to_celsius(50))

مقادیر بازگشتی

توابع می‌توانند با استفاده از دستور return داده‌ها را به کدی که آن‌ها را فراخوانی کرده است برگردانند.

وقتی یک تابع به یک دستور return می‌رسد، اجرا را متوقف می‌کند و نتیجه را برمی‌گرداند:

مثال
def get_greeting():
  return "Hello from a function"

message = get_greeting()
print(message)

شما می‌توانید از مقدار بازگشتی مستقیماً استفاده کنید:

مثال
def get_greeting():
  return "Hello from a function"

print(get_greeting())

اگر یک تابع دستور return نداشته باشد، به طور پیش‌فرض None را برمی‌گرداند.

دستور pass

تعاریف تابع نمی‌توانند خالی باشند. اگر نیاز دارید یک نگهدارنده تابع بدون هیچ کدی ایجاد کنید، از دستور pass استفاده کنید:

مثال
def my_function():
  pass

دستور pass اغلب هنگام توسعه استفاده می‌شود، که به شما اجازه می‌دهد ابتدا ساختار را تعریف کنید و جزئیات را بعداً پیاده‌سازی کنید.

آزمون: توابع پایتون

برای سنجش درک خود از مفاهیم توابع در پایتون، به سوالات زیر پاسخ دهید:

سوال 1: کدام کلمه کلیدی برای تعریف یک تابع در پایتون استفاده می‌شود؟

سوال 2: کدام دستور برای برگرداندن یک مقدار از یک تابع استفاده می‌شود؟

سوال 3: اگر یک تابع هیچ دستور return نداشته باشد، چه چیزی را برمی‌گرداند؟
