بولین پایتون

بولین‌ها یکی از دو مقدار را نشان می‌دهند: True یا False.

مقادیر بولین

در برنامه‌نویسی، اغلب نیاز دارید بدانید که یک عبارت True است یا False.

شما می‌توانید هر عبارتی را در پایتون ارزیابی کنید و یکی از دو پاسخ True یا False را دریافت کنید.

وقتی دو مقدار را با هم مقایسه می‌کنید، عبارت ارزیابی شده و پایتون پاسخ بولین را برمی‌گرداند:

print(10 > 9)
print(10 == 9)
print(10 < 9)
خروجی:
True
False
False

وقتی یک شرط را در یک عبارت if اجرا می‌کنید، پایتون True یا False را برمی‌گرداند:

a = 200
b = 33

if b > a:
  print("b is greater than a")
else:
  print("b is not greater than a")
خروجی:
b is not greater than a

ارزیابی مقادیر و متغیرها

تابع bool() به شما امکان می‌دهد هر مقداری را ارزیابی کنید و در ازای آن True یا False را به شما بدهد.

print(bool("Hello"))
print(bool(15))
خروجی:
True
True
x = "Hello"
y = 15

print(bool(x))
print(bool(y))
خروجی:
True
True

اکثر مقادیر True هستند

تقریباً هر مقداری که محتوایی داشته باشد، به True ارزیابی می‌شود.

هر رشته‌ای True است، به جز رشته‌های خالی.

هر عددی True است، به جز 0.

هر لیست، تاپل، مجموعه و دیکشنری True است، به جز موارد خالی.

bool("abc")
bool(123)
bool(["apple", "cherry", "banana"])
خروجی:
True
True
True

برخی مقادیر False هستند

در واقع، مقادیر زیادی وجود ندارد که به False ارزیابی شوند، به جز مقادیر خالی، مانند ()، []، {}، ""، عدد 0 و مقدار None. و البته مقدار False به False ارزیابی می‌شود.

bool(False)
bool(None)
bool(0)
bool("")
bool(())
bool([])
bool({})
خروجی:
False
False
False
False
False
False
False

یک مقدار دیگر، یا شیء در این مورد، به False ارزیابی می‌شود، و آن این است که اگر شما یک شیء داشته باشید که از یک کلاس با تابع __len__ ساخته شده باشد که 0 یا False را برمی‌گرداند:

class myclass():
  def __len__(self):
    return 0

myobj = myclass()
print(bool(myobj))
خروجی:
False

توابع می‌توانند یک بولین برگردانند

شما می‌توانید توابعی ایجاد کنید که یک مقدار بولین برگردانند:

def myFunction():
  return True

print(myFunction())
خروجی:
True

شما می‌توانید کد را بر اساس پاسخ بولین یک تابع اجرا کنید:

def myFunction():
  return True

if myFunction():
  print("YES!")
else:
  print("NO!")
خروجی:
YES!

پایتون همچنین توابع داخلی زیادی دارد که یک مقدار بولین برمی‌گردانند، مانند تابع isinstance() که می‌تواند برای تعیین اینکه آیا یک شیء از نوع داده خاصی است استفاده شود:

x = 200
print(isinstance(x, int))
خروجی:
True

آزمون درک مطلب

1. کدام یک از مقادیر زیر به False ارزیابی می‌شود؟

2. خروجی کد زیر چیست؟

x = 0
print(bool(x))
